Mantis shrimp are dangerous

Mantis shrimps have powerful raptorials that are used to attack and kill prey either by spearing, stunning, or dismembering. Some mantis shrimp species have specialised calcified "clubs" that can strike with great power, while others have sharp forelimbs used to seize the prey
